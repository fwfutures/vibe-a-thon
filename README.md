# Vibe-a-thon Skills

Agent skills for the Vibe-a-thon. Install as a Claude Code plugin for instant access to all skills.

## Install

```bash
npx skills add fwfutures/vibe-a-thon -g
```

## Skills

| Skill | Description |
|-------|-------------|
| frontend-design | Create distinctive, production-grade frontend interfaces with high design quality |
| gcp-deploy | Deploy containerised applications to Google Cloud Run from source using gcloud CLI |
| humanizer | Remove signs of AI-generated writing from text |
| agent-browser | Browser automation CLI for AI agents |
| find-skills | Discover and install agent skills from the open ecosystem |
| skill-development | Guidance on creating effective skills for Claude Code plugins |
| setup-windows-dev | Set up a fresh Windows machine for Node.js development |
